Self-typing typewriter

As part of a big art installation showcasing the ongoing race between artificial intelligence and humanity, I was tasked to create a self-
typing typewriter.

I tackled this task using electromagnetic
solenoids, each utilizing an activation device
pulling on its respective key. This solenoid
was connected to relays that allowed for
quick and safe control of the critical press
timings using the digital output pins of a
microcontroller. Furthermore, multiple servos
were used to control the shift key or the
paper's vertical position when moving a lever.
The system also consisted of a stepper motor
used to control the linear position of the
carriage. The whole system was hidden
inside a wooden table, making only the
typewriter visible.
The system was programmed to receive text
digitally and then convert it into a sequence
for the specific solenoid activation order.
